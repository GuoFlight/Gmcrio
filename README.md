# 关于项目

> 微服务Demo程序

# 特点

配置文件

* 使用toml格式

日志

* 每个level都会独立输出到不同的文件。
* 日志切割。
* 支持输出文件名、函数名和行号。
* 非DEBUG模式不输出到终端。

http服务器

* 采用Gin框架。
* 支持日常请求记录日志。

# Question

问：为什么要自己实现一个输出行号的Hook？
答：因为logrus作者也不建议输出行号，这会降低程序的性能。这也是logrus迟迟不增加这项功能的原因。而我的Hook在INFO这个level上是不打日志的，所以性能的损耗可以忽略。
